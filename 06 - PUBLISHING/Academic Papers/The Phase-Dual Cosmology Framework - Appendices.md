## Appendix A: Glossary of Terms

### Core Concepts

**Phase-Dual Cosmology (PDCC)**  
A cosmological framework proposing that our observable universe exists alongside a gravitationally-coupled but electromagnetically-isolated "Phase Universe," separated by a dynamic Φ-boundary field. Both universes were created simultaneously through a collision event between pre-existing Origin Black Holes and Phase Origin Black Holes.

**Phase Universe**  
The mirror universe existing in the (T+δt, X+δx) reference frame, gravitationally coupled to our universe (T, X) through the Φ-boundary. Contains antimatter-dominated structure with inverted matter/antimatter ratio (1:10 vs our 10:1).

**Φ-Boundary (Phi-Boundary)**  
The dynamic elastic boundary field separating our universe from the Phase Universe. Acts as a membrane allowing gravitational coupling while maintaining electromagnetic isolation. Characterized by variable separation distance θ(x) that determines local gravitational coupling strength.

**θ(x) - Boundary Separation Function**  
The spatially-varying distance between phase-separated universes. Small θ(x) → strong gravitational coupling (mimics dark matter). Large θ(x) → weak coupling (mimics low dark matter regions). Ranges from θ_min in voids to θ_max in superclusters.

**G_eff(x) - Effective Gravitational Coupling**  
G_eff(x) = G · (1 + α/θ(x))  
The spatially-varying effective gravitational constant arising from Φ-boundary coupling. Explains rotation curves, gravitational lensing, and cluster dynamics without requiring dark matter particles.

**Phase Tension (T_static)**  
The elastic tension in the Φ-boundary arising from phase field curvature:  
T_static(x) = k · |∇Φ|²  
Creates restoring force that maintains boundary integrity and generates harvestable energy in regions of high curvature.

### Entities and Structures

**Origin Black Holes (OBH)**  
Massive black holes (10^6 to 10^9 M☉) that existed before the Big Bang collision event and survived to seed early structure formation. Explain JWST observations of supermassive black holes at z > 10. Characterized by:

**Cascade Black Holes**  
Black holes formed during secondary collision events in the cascade propagation phase. Characterized by:
- Mass range 9-10 M☉ (light seeds)

- Less common than surviving OBHs

**Phase Black Holes (PBH)**  
Black holes in the Phase Universe. Their gravitational influence leaks through the Φ-boundary, contributing to observed dark matter effects.

def initialize_collision(phi_field, collision_energy, position, spread):
    """
    Initialize collision-induced phase field component.
    
    Parameters:
    - phi_field: 3D array for phase field
    - collision_energy: Energy of collision (in 10⁶⁹ J)
    - position: (x, y, z) collision center
    - spread: Gaussian spread parameter (in Mpc)
    """
    gx, gy, gz = np.meshgrid(
        np.arange(grid_size),
        np.arange(grid_size),
        np.arange(grid_size),
        indexing='ij'
    )
    
    # Distance from collision center
    r = np.sqrt(
        ((gx - position[0]) * dx)**2 +
        ((gy - position[1]) * dx)**2 +
        ((gz - position[2]) * dx)**2
    )
    
    # Collision-induced phase perturbation (Gaussian)
    phi_coll = collision_energy * np.exp(-(r / spread)**2)
    
    return phi_field + phi_coll

def cascade_collisions(phi_field, n_cascades=5):
    """
    Generate cascade of secondary collisions.
    
    Parameters:
    - phi_field: Initial phase field
    - n_cascades: Number of cascade generations
    
    Returns:
    - Updated phi_field with cascade structure
    - List of void properties (position, size, energy)

    voids = []
    current_energy = primary_energy
    
    for generation in range(n_cascades):
        # Energy decreases with each generation
        current_energy *= cascade_retention
        
        # Random positions for secondary collisions
        n_collisions = 2**(generation + 1)  # Exponential increase
        
        for i in range(n_collisions):
            # Random position biased toward primary void edges
            theta = np.random.uniform(0, 2*np.pi)
            phi_angle = np.random.uniform(0, np.pi)
            r_offset = 100 * (1 + generation * 0.5)  # Mpc from center
            
            position = (
                grid_size//2 + int(r_offset * np.sin(phi_angle) * np.cos(theta) / dx),
                grid_size//2 + int(r_offset * np.sin(phi_angle) * np.sin(theta) / dx),
                grid_size//2 + int(r_offset * np.cos(phi_angle) / dx)
            )
            
            # Collision spread decreases with generation
            spread = primary_spread * (cascade_retention ** (generation + 1))
            
            phi_field = initialize_collision(
                phi_field, current_energy, position, spread
            )
            
            # Record void properties
            voids.append({
                'position': position,
                'size': spread * 2,  # Diameter
                'energy': current_energy,
                'generation': generation
            })
    
    return phi_field, voids

#### C.2.4 Boundary Separation Calculation

```python
def calculate_delta_field(phi_field):
    """
    Calculate boundary separation θ(x) from phase field.
    
    θ(x) = θ_min + (θ_max - θ_min) * f(Φ(x))
    
    where f is smoothing function.
    """
    # Normalize phase field
    phi_normalized = (phi_field - phi_field.min()) / (phi_field.max() - phi_field.min())
    
    # Threshold and scale for tanh smoothing
    phi_threshold = 0.5
    phi_scale = 0.1
    
    # Smoothing function
    f = 0.5 * (1 + np.tanh((phi_normalized - phi_threshold) / phi_scale))
    
    # Calculate θ(x)
    delta_field = delta_min + (delta_max - delta_min) * f
    
    return delta_field
```

def analyze_simulation_ensemble(all_results):
